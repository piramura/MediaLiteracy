# 📡 モールス信号 メディアリテラシー体験アプリ

モールス信号の仕組みを学習・体験できるインタラクティブなWebアプリケーションです。テキストとモールス信号の相互変換、音声再生、MP3解析、クイズ、資料閲覧などを通じてメディアコミュニケーションの歴史と技術を理解できます。

対応言語: 日本語（ひらがな）、日本語（ローマ字）、English
キッズモード: あり（ひらがな中心の優しい表記）
##  主な機能

| 機能 | 説明 |
|------|------|
| **テキスト変換** | 日本語（ひらがな）↔ モールス信号（・−）の相互変換 |
| **音声再生** | モールス信号を音声（ビープ音）で再生 |
| **MP3解析** | モールス信号を含むMP3ファイルをアップロード・解析 |
| **モールス信号表** | 各文字・数字のモールス表記を参照 |
| **クイズ機能** | モールス記号からひらがなへの変換学習 |
| **SNSシェア** | 学習成果をSNSで共有 |

## 使用方法

### 基本的な変換・再生
1. **「はじめる」をクリック**してメイン画面へ進む
2. **テキストを入力**し「→ 名前を決定！」をクリック
3. モールス信号が表示されたら「🔊 再生」で音声確認・ダウンロード可
4. モールス信号を直接入力して「→ 答え合わせ」
5. クイズに挑戦

### MP3ファイルの解析
1. 「モールス音声(MP3)をアップロードして解析」からファイル選択、またはモールスをコピペで入力、またはボタンで入力
2. 検出されたモールス記号がテキスト欄に出力
3. 日本語(ローマ字)の場合、ローマ字に対応するひらがなも表示

### クイズ
1. 最後までいく
2. モールス信号が表示される
3. 対応する言葉を選択して解答
4. 結果をSNSでシェア可能

### モールス信号表の確認
1. 「モールス信号表」タブを開く
2. 各文字・数字・記号のモールス表記を確認
3. 使用言語に対応する文字が欧文コードの時のみ、Playボタンが出現
4. 音の覚え方を確認できる

## 🔊 MP3解析機能の詳細

### 仕組み
- **WebAudio API** でMP3をデコード
- **RMS（二乗平均平方根）** で音声信号の包絡線を計算
- **自動閾値推定** でモールス信号のオン/オフを判定
- **時間解析** で「・」(dot: 短音)と「−」(dash: 長音)を分類
- **無音区間** で文字間（`／`）および語間（スペース）を判定

### 精度について
- 品質の良い音声ファイル（背景雑音が少ない）ほど解析精度が高い
- ノイズが多い場合は正確に解析できない可能性がある
- 「ー」単独だと「・」と認識してしまう

## 👥 開発・拡張ガイド
- 「・」「ー」「／」をユーザーが自由に変更できるとよい
- ボタンを押す長さでモールス入力
- 新しい言語（フランス語など）の追加

### ファイル構成と役割

| ファイル | 役割 | 拡張ポイント |
|---------|------|-----------|
| `converter.js` | モールス信号エンコード・デコード |「・」など自由に設定できるように変更|
| `ui.js` | 画面表示・ユーザーインタラクション | UI要素の追加、イベント処理の追加 |
| `audio-player.js` | モールスの音を再生 | 矩形波などsin波以外の音を追加 |
| `constants.js` | 定数 | より使いやすいように定数をまとめる |
| `changeLanguage.js` | 言語変換 | より対応個所をわかりやすく |
| `settings.js` | 定数設定機能 | 機能追加 |
| `quiz.js` | クイズ機能 | 問題セットの拡張、採点ロジックの改善 |
| `style.css` | デザイン・レイアウト | テーマ変更、レスポンシブ対応の改善 |
